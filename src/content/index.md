## How to Use This Course

1. **Complete modules in order.** Each module builds on the previous one.
2. **Interact with 3-D models.** Click any tool or pipe assembly in the viewer to trigger
   its animation and read the annotated callouts.
3. **Take every quiz before advancing.** A passing score of **70 %** is required to unlock
   the next module.
4. **Review the midterm before Module 5.** The midterm reinforces foundational knowledge
   critical to DWV and fixture work.
5. **Earn your completion certificate** by passing the final exam with ≥ 75 %.

## Learning Objectives — Complete Course

By the end of this course, learners will be able to:

- Describe the history of plumbing and the structure of the plumbing trade
- Identify, select, and safely operate the standard tools of the trade
- Specify appropriate pipe materials and fittings for a given application
- Explain how potable water is delivered under pressure from the main to the fixture
- Size and install a code-compliant DWV system including traps and vents
- Install common residential fixtures and water-heating appliances
- Read a basic plumbing plan and locate relevant code sections
- Apply OSHA-required safety practices on a plumbing job site

## Prerequisites

- Basic arithmetic (fractions, percentages, unit conversions)
- Comfort using hand tools
- No prior trade experience required
