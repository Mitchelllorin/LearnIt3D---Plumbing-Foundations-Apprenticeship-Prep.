## Lesson 7.1 — Construction Document Hierarchy

A complete set of construction drawings (also called plans, blueprints, or CD sets) for a plumbing project includes multiple drawing disciplines. As a plumber, you will work primarily with the Plumbing drawings (prefix **P**) but must coordinate with:

| Drawing Prefix | Discipline | Relevance to Plumber |
|---|---|---|
| A | Architectural | Room layout, wall thickness, finish floor elevations |
| S | Structural | Beam and floor locations — cannot cut without engineer approval |
| M | Mechanical (HVAC) | Conflicts in ceiling/wall space |
| P | Plumbing | Primary working drawings |
| E | Electrical | Coordination — shared utility chases |
| C | Civil / Site | Utility connections, grading, septic |

**Sheet numbering example:**
- P-001 — Plumbing symbols legend and general notes
- P-101 — First floor plumbing plan
- P-201 — Second floor plumbing plan
- P-501 — Plumbing riser diagrams
- P-601 — Plumbing details

## Lesson 7.2 — Plumbing Plan Reading

### Scale

Plumbing floor plans are typically drawn at **1/8" = 1'-0"** or **1/4" = 1'-0"** for residential, and up to 1/16" = 1'-0" for large commercial projects. Always verify the scale printed on the drawing before using a scale rule — plans are sometimes reproduced at a reduced size.

### Common Plumbing Symbols

**Diagram placeholder:** `[FIGURE 7-1: Plumbing symbol legend — 3-D interactive sheet; hover over each symbol to see its name]`

#### Supply Piping Symbols (typically shown as lines):

| Symbol | Meaning |
|---|---|
| ──── (solid thin line) | Cold water supply (CW) |
| ─ ─ ─ (dashed line) | Hot water supply (HWS) |
| ─ · ─ · ─ (dash-dot) | Hot water return (HWR) |
| ─ ─ G ─ ─ | Gas supply |
| ─ ─ F ─ ─ | Fire suppression / sprinkler |

#### DWV Piping Symbols:

| Symbol | Meaning |
|---|---|
| ──── (medium solid line) | Drain / waste pipe |
| ─ ─ ─ (medium dashed) | Vent pipe |
| ──── (heavy line) | Building drain / sewer |

#### Fixture Symbols (plan view):

| Symbol | Fixture |
|---|---|
| Rectangle with oval | Water closet (toilet) |
| Rectangle | Lavatory or sink |
| Large rectangle | Bathtub |
| Small square with circle | Floor drain |
| Circle with lines | Showerhead |
| Circle | Cleanout (CO) |

### Reading a Floor Plan

When reviewing a plumbing plan, follow this sequence:
1. **Locate all fixtures** in each room.
2. **Trace the supply lines** from the main to each fixture; note pipe sizes (written adjacent to the line or in a schedule).
3. **Trace the DWV** from each fixture back to the stack and building drain.
4. **Identify all cleanouts** and verify locations comply with code.
5. **Note all special items:** water meters, PRVs, backflow preventers, water heaters, hose bibs, irrigation.

## Lesson 7.3 — Plumbing Riser Diagrams

A **riser diagram** (also called a schematic diagram) is a vertical representation of the DWV system — typically drawn in two-dimensional isometric or orthographic projection. It shows all stacks, branches, fixtures, and vent connections on every floor simultaneously.

**Diagram placeholder:** `[FIGURE 7-2: 3-story residential DWV riser diagram — 3-D interactive; click any pipe segment to show IPC code reference]`

### How to Read a Riser Diagram

1. **Start at the building sewer** (bottom).
2. Trace upward through the building drain, stacks, and branch drains.
3. Note the fixture connections at each floor.
4. Trace the vent pipes upward to the stack vent penetrating the roof.
5. Note pipe sizes on each segment — sizes decrease as fewer fixtures are connected upstream.

### Isometric Drawings

An **isometric** is a three-dimensional representation drawn at 30° angles. It shows the spatial arrangement of piping in a way that a flat floor plan cannot. Isometrics are commonly used for:
- Water supply rough-in layouts
- DWV system coordination in complex buildings
- Shop drawings for prefabricated pipe assemblies

## Lesson 7.4 — Using the Plumbing Code

### Code Structure

Both the IPC and UPC are organized similarly:
- **Chapters:** Organized by system (administration, definitions, general regulations, fixtures, water supply, sanitary drainage, vents, storm drainage, fuel gas, etc.)
- **Sections:** Numbered sequentially within each chapter (e.g., IPC 704.1 — Horizontal drainage piping — slope)
- **Exception clauses:** Modifications to the general rule for specific conditions
- **Tables:** Quick-reference values (pipe sizing, fixture units, slopes)
- **Appendices:** Informational or optional provisions — adopted only if specifically referenced in the local adopting ordinance

### How to Research a Code Question

**Example:** *"Does a 4" horizontal drain require ¼" per foot slope or can I use 1/8"?"*

Step 1: Look in the **Index** for "Drainage, horizontal — slope" or "Slope, drain pipe."
Step 2: Turn to the referenced section (IPC 704.1).
Step 3: Read the full section including exceptions.
Step 4: Check any referenced tables for the specific pipe size.
Step 5: Check if the local jurisdiction has any amendments to the adopted code.

> 🔑 **The specific rule always overrides the general rule.** If a general section says one thing and a more specific section covering your exact situation says another, follow the specific section.

### Code Hierarchy

```
Federal Law / EPA Regulations (must comply)
        ↓
State Law / State Plumbing Board (may be stricter)
        ↓
Adopted Model Code (IPC or UPC as modified by state/local amendments)
        ↓
Local Amendments (city or county modifications to the model code)
        ↓
Manufacturer's Instructions (must comply with code AND instructions)
```

## Lesson 7.5 — Permits and Inspections

### Why Permits Exist

The permit process exists to ensure:
- Work is performed by licensed contractors
- Work is inspected by a trained official before it is concealed in walls
- The public record shows the work was performed and approved

### The Permit Process

**Diagram placeholder:** `[FIGURE 7-3: Permit workflow diagram — 3-D flowchart from application through final approval]`

1. **Application:** The licensed contractor (or homeowner in many jurisdictions) submits plans and a permit application to the local Building Department (also called the Building Division, Authority Having Jurisdiction — AHJ, or inspection authority).
2. **Plan review:** For large projects, plans may be reviewed by a plans examiner before permit issuance. Residential permits are often over-the-counter.
3. **Permit issuance:** The permit card must be posted at the job site.
4. **Rough-in inspection:** Before covering any pipes in walls, floors, or ceilings, the plumber calls for a rough-in (or underground) inspection. The inspector verifies pipe sizes, slopes, venting, and test results.
5. **Pressure / flood test:** The inspector typically requires that the system be under test at the time of rough-in inspection (10 psi air test for supply; flood or air test for DWV).
6. **Final inspection:** After fixtures are installed and the system is operational, the final plumbing inspection is requested.
7. **Certificate of Occupancy (CO):** The building cannot be legally occupied until all trades have passed final inspection.

### Testing Requirements

**Water supply test (IPC 312.5):**
- Hydrostatic: 1.5 × working pressure or minimum 100 psi for 15 minutes
- Air test: 50 psi for 15 minutes (residential)

**DWV test (IPC 312.2–312.3):**
- Water test: Fill all pipes to the top of the highest stack or to the rim of the highest fixture rough-in for 15 minutes — no drop in level
- Air test: 5 psi for 15 minutes

## Lesson 7.6 — Common Code Violations

Understanding common code violations helps you avoid them on the job:

| Violation | Code Reference | How to Avoid |
|---|---|---|
| Trap arm exceeds maximum length | IPC Table 909.1 | Measure and vent within allowed distance |
| No cleanout at base of stack | IPC 708.3.5 | Install cleanout fitting in rough-in |
| Improper support spacing | IPC Table 308.5 | Follow hanger spacing table for each pipe material |
| Vent pipe not through roof | IPC 903 | Route all required vents to open air |
| P-trap below the weir (S-trap) | IPC 1002 | Verify trap arm exits horizontally, not downward |
| PRV missing on high-pressure supply | IPC 604.8 | Check static pressure; install PRV if > 80 psi |
| T&P valve discharge not piped to floor | IPC 504.6 | Always pipe T&P discharge to within 6" of floor |

## Module 7 Summary

- Construction documents include Architectural, Structural, Mechanical, Plumbing, Electrical, and Civil drawings
- Plumbing plans use standardized symbols for supply, DWV, and fixtures
- Riser diagrams show the vertical DWV arrangement from sewer to stack vent
- Use the code index → section → exceptions → tables workflow to research code questions
- The permit process: application → plan review → permit → rough-in inspection → final inspection → CO
- Always test systems (pressure or flood) before calling for inspection
