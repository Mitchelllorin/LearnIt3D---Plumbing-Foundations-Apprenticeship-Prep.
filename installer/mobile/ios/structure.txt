=======================================================
Build outputs
  Debug IPA   : build/Debug-iphoneos/LearnIt3DPlumbing.app
  Release IPA : build/LearnIt3DPlumbing.ipa   (via xcodebuild archive)

Minimum device requirements
  OS         : iOS 16.0
  Device     : iPhone X or later / iPad Air (3rd gen) or later
  Storage    : 250 MB free
  GPU        : Metal support required (for 3-D scenes)

Capabilities required (Xcode Signing & Capabilities)
  • Push Notifications
  • Background Modes → Background fetch (content updates)
  • In-App Purchase   (future module unlock)
  • iCloud            (optional: CloudKit progress sync)

Distribution channels
  1. App Store (primary)
  2. TestFlight (beta / apprenticeship programme testers)
  3. Ad-hoc / MDM (enterprise deployment to trade schools)
